Editing fonts using CSS

CSS Font Properties

Property Description Example Declaration
font-family specifies a prioritized list of one or more font family names and/or generic family names for the selected element .element {
font-family: Garamond, Georgia, serif;
}
font-weight determines the thickness of the font. Can be expressed in words such as 'light', 'normal', 'bold', 'extra-bold', or numerically in multiples of 100 up to 900. .element {
font-weight: 900;
}
font-style sets whether a font should be styled normal or in italics .element {
font-style: italic;
}
font-size Sets the font size of a given element. This can be set in px, absolutes, relatives, percentages and Ems. Visit MDn 'font-size' for a thorough breakdown on potential values .element {
font-size: 2em;
}
text-transform specifies how to capitalize an element's text. It can be used to make text appear in all-uppercase or all-lowercase, or with each word capitalized .element {
text-transform: uppercase;
}

Adjusting spacing

Property Description Example Declaration
line-height Used to alter the vertical spacing between lines of text. Unitless number is the preffered syntax for this declaration .element {
line-height: 2.5;
}
word-spacing Changes how far apart individual words are using the term 'normal', percentages, px (preferred), or em .element {
word-spacing: 14px;
}
letter-spacing Changes how far apart individual letters are using the term 'normal', px, or em .element {
letter-spacing: 1em;
}
